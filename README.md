# TypeScript_Basics
Apprentissage en autonomie du TypeScript

## Pourquoi ?
Après plus de 3 ans ne faire que de la programmation orienté scripting et backend, je veux sortir de ma zone de confort : le WEB.

Le TypeScript n'est pas un langage qui m'est inconnu : que ce soit Notion, DocMost ou bien Streamlit, j'ai pu constater son utilité et ses capacités.

## Liste des notions vu :
- Les types ;
- Les objets ;
- Les fonctions (paramètre optionnel) ;
- Les interfaces (déclaration + usage) ;
- Les types personnalisés ;
- Les conditions ;
- Les boucles (for et for of) ;
- Les Classes (déclaration + usage) ;
